NOTE: Please set the Google Map API and the Google AJAX API key before use of software.

How to obtain Google Map API and Google AJAX API keys

  1. Open web browser and navagate to: http://www.google.com/apis/maps/signup.html
  2. Scroll down and enter website URL to software loction. E.g. http://www.example.com/installdir/
  3. Click Generate API Key
  4. Write down generated API key
  5. Navagate web browser to: http://code.google.com/apis/ajaxsearch/signup.html
  6. Scroll down and enter webset URL to software location. E.g. http://www.example.com/installdir/
  7. Click Generate API Key
  8. Write down generated API key
  9. Continue to instructions below

How to install Google Map API and Google AJAX API keys

  1. Login to PHP Store Locator administration panel.

    Login URL: http://www.example.com/installdir/sladmin/index.php
    Default Username: admin
    Default Password: admin

  2. Click General Settings Link
  3. Enter Google Map API Key in Google Map API text box
  4. Enter Google AJAX API Key in Google AJAX API text box
  5. Click Update Settings to save
